DEVOPS TOOL: TERRAFORM

Infrastructure as Code (IaC)

What is Terraform?

  • 📦 Open source Infrastructure as Code (IaC) tool by HashiCorp
  • 📝 Defines and provisions infrastructure using declarative config files
  • ☁️ Supports AWS, Azure, Google Cloud & many providers
  • 🔄 Manages entire infrastructure lifecycle efficiently
  • ⚡ Enables automation & version control for deployments

How It Works

resource "aws_instance" "Avishkar" {
  ami = "ami-08982f1c5bf93d976"
  instance_type = "t3.micro"
  key_name = "test"
}

Terraform Workflow

Follow these main steps to manage infrastructure:

⚙️ Init 📝 Plan 🚀 Apply 🗑️ Destroy

Key Benefits

  • ✅ Efficiency: Automates resource provisioning & reduces manual work
  • 🔒 Consistency: Avoids human errors & ensures predictable deployments
  • ♻️ Immutable Infra: Prevents configuration drift & improves stability
  • 🤝 Collaboration: Works with Git for teamwork, reviews, and rollbacks

Core Concepts

  • 📦 Resource: Infrastructure object managed by Terraform
  • ☁️ Provider: Services like aws_s3_bucket, google_compute_instance
  • 📚 Module: Reusable building block of configurations
  • 🔗 Data Sources: Manage complex dependencies dynamically

Example Use Cases

  • ☁️ Cloud Resource Provisioning: Launch VMs, DBs, networks in minutes
  • 🌐 Multi-Cloud Deployments: Manage AWS, Azure, and GCP together
  • 🐳 Container Orchestration: Deploy & scale Kubernetes clusters
  • 🧪 Dev/Test Environments: Quickly spin up isolated test setups
