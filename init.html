Terraform: Init

Initialize Your Terraform Project

What is terraform init?

The terraform init command initializes a working directory containing Terraform configuration files. It downloads and installs the required provider plugins, sets up the backend for state storage, and prepares your environment for Terraform operations.

When to Use It?

Run terraform init once at the beginning of a new or existing project before any other Terraform commands. You also rerun it if you modify backend configuration or add new providers.

Command

terraform init
